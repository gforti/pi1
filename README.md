# LED Raspberry PI Testing

## Installation

```sh
$ sudo npm install --unsafe-perm
```

## Issues with NPM install

```sh
$ sudo npm rebuild
```

## Stop the App
```sh
$ ctrl + c
```

## Shutdown PI
```sh
$ sudo shutdown -h now
```

## Restart PI
```sh
$ sudo shutdown -r now
```


## Running the App

 Note that everyone must be on the same network/WiFi. `Chrome` is the recommend web browser.

> `Will run on `http://[ip_address]:3000/`
